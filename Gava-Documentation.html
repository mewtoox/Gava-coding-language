Gava-Documentation

printing to console in Gava

printn("hello world");//prints hello world to console

Turning text to binary in Gava

store("hello world");//turns hello world in binary:0110100001100101011011000110110001101111001000000111011101101111011100100110110001100100

Doing operations in Gava

add(1,2);//1+2

subtract(1,2);//1-2

multiply(1,2);//1*2

divide(1,2);// 1/2

Keep printing hashtag in Gava

loophashtag(2);//This prints: ##

Making a canvas in Gava

canvas(1);//prints a 1 by 1 hashtag(#) so for example it prints this:#

Checking if code works in Gava

check();//prints checker in the console so then you can check where your code breaks if there are errors

Random numbers in Gava

random(1);//prints a random whole number 0 to 1.
